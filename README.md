# Citi Bike Data Analysis

## Project Overview
Exploratory data analysis of Citi Bike trip data using Apache Spark DataFrame API, including usage patterns and weather impact analysis.

## Features
- Data cleaning and validation
- Feature engineering (duration, time features, round trips)
- Ride pattern analysis by time and user type
- Station and route analysis
- Integration with Meteostat weather data

## Analysis
- Hourly and weekly usage patterns
- Member vs casual user behavior
- Top stations and routes
- Impact of weather conditions on rides

## Output
- Aggregated datasets
- Analytical tables
- Exported results (CSV/JSON)
